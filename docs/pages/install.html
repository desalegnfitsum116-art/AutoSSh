Installation

Linux

Debian / Ubuntu (.deb)

sudo dpkg -i auto-ssh_0.1.0_amd64.deb

AppImage (all distros)

chmod +x auto-ssh-x86_64.AppImage
./auto-ssh-x86_64.AppImage

From source

git clone https://github.com/yourusername/autossh.git
cd autossh
cargo build --release
strip target/release/auto-ssh
./target/release/auto-ssh

System Dependencies

sudo apt install build-essential libssl-dev cmake

The system tray uses a pure Rust StatusNotifierItem implementation — no additional indicator libraries required.
