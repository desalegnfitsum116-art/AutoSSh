Contributing

Contributions are welcome! AutoSSH is a small utility built by people who want SSH connections to be boring — set once, connect forever.

Development Setup

git clone https://github.com/yourusername/autossh.git
cd autossh
cargo build

Code Quality

cargo fmt              # Format code
cargo clippy --all-targets --all-features  # Lint
cargo test             # Run tests
cargo check            # Fast compilation check

Pull Request Checklist

  • cargo fmt has been run
  • cargo clippy passes with no warnings
  • cargo test passes
  • cargo build --release compiles successfully
  • Changes include appropriate error handling and logging

Branch Naming

feature/multiple-devices
fix/tray-icon-crash
docs/readme-typo
refactor/monitor-module

Commit Messages

Follow Conventional Commits:

feat: add automatic reconnection with exponential backoff
fix: handle SSH key path with tilde expansion
docs: update configuration example in README
refactor: extract network check into separate module
chore: bump tokio dependency to 1.40

See CONTRIBUTING.md on GitHub for the full guide.
