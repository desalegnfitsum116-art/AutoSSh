Changelog

v0.1.0 — Initial Release

Features

  • Automatic SSH connection when remote device becomes reachable
  • Automatic reconnection with exponential backoff (1s → 2s → 4s → 8s → 16s → 32s → 60s caps)
  • SSH key authentication via libssh2 (Ed25519, RSA)
  • Background network monitoring (DNS resolution + TCP port check)
  • Live status dashboard with color-coded device states
  • Dark-themed native GUI (egui/eframe)
  • System tray integration (StatusNotifierItem via DBus)
  • Session keepalive (every 15s) and health checks (every 30s)
  • Session recovery after sleep/wake and network interruptions
  • TOML configuration at ~/.config/auto-ssh/config.toml
  • XDG autostart support (start on boot toggle)
  • Minimize to tray on close
  • Comprehensive logging via env_logger (RUST_LOG=info)

Packages

  • Debian/Ubuntu .deb package (5.1 MB)
  • Portable AppImage (6.7 MB)
  • Stripped Linux ELF binary (17 MB)

Notes

  • Single remote device support (multiple devices planned)
  • Linux x86_64 only (cross-platform support planned)
  • SSH key authentication only (password auth not supported)
  • No host key fingerprint verification yet
