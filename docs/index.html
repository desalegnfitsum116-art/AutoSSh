Introduction

AutoSSH is a lightweight desktop utility that automatically connects trusted devices through SSH when both devices are powered on and reachable on the network. No terminals, no manual commands.

Rust MIT License Cross Platform v0.1.0

Features

  • Auto-connect — Establishes an SSH session as soon as the remote device becomes reachable
  • Auto-reconnect — Survives network drops, sleep/wake cycles, and remote reboots with exponential backoff
  • Zero terminal usage — No need to open a shell and type SSH commands
  • SSH key authentication — Uses your existing Ed25519 or RSA keys
  • Background service — Runs quietly in the system tray and monitors devices 24/7
  • Live status — Dashboard shows Offline → Online → SSH Ready → Connected
  • System tray integration — Connect, disconnect, or quit directly from the tray menu
  • Native GUI — Built with egui/eframe — lightweight, dark-themed
  • Start on boot — Optionally launch at login and minimize to tray
  • Portable config — Settings stored in TOML format at ~/.config/auto-ssh/config.toml

Use Cases

  • Developers who want to SSH into a home server without typing ssh user@host every time
  • Homelab users who need persistent terminal access to a Raspberry Pi
  • DevOps engineers managing jump-box access
  • Anyone managing multiple computers who wants SSH connections that just work
